# Fake News Detection System

This is an AI-based Fake News Detection System developed using Machine Learning and NLP techniques.
The system classifies news as real or fake based on textual content.

## Features
- Fake vs Real news classification
- NLP-based text preprocessing
- Machine Learning model (Naive Bayes)
- Simple user interface using Streamlit

## Tech Stack
- Python
- Scikit-learn
- NLP
- Streamlit

## How to Run the Project
1. Clone the repository
2. Install required libraries
3. Run the Streamlit app
